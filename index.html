Cryptographic failures Demo

No Cryptography (Unsafe)

⚠️ Password is stored in plain text - anyone can read it!
This might seem silly but this is what happened with Exactus

With Hashing (Better)

🔒 Password hashed - same passwords have same hashes and is easily cracked with a Rainbow table!

Hash Cracking

🔓 Hashed passwords are very easy to break with a rainbow table

With Salting (Best)

🔒🔑 Password is hashed with unique salt - even same passwords look different!

Weak Key Management (Misconfiguration)

⚠️ This crytpographic failure myuses a hardcoded encryption key! Attackers who find the key can decrypt any data.
